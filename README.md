# Spotify Checker
> This is my first Python-made bruteforcer. I used to made them in C# before. Python is surely not the best because of its "fake" multithreading but it provide a very good HTTP support with a very small rate of errors and also a very pleasant cross-platform support.

## Requirements
- Python 3 or greater
- Having [Click](http://click.palletsprojects.com/) library

## Usage
```
Usage: checker.py [OPTIONS] INPUT

Options:
  --threads INTEGER  Number of threads for the task (default: 4)
  --output TEXT      Output file (e.g.: output.txt)
  --help             Show this message and exit.
```
